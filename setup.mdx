The setup page makes lists all the required info needed to complete and submit your store for review. You can still manage those settings from various settings in the app.

<Steps>
  <Step title="Customize your store">
    Add your store logo, support email, and set your refund policy
  </Step>
  <Step title="Add your first product">
    Simple products are standalone items with no options, featuring only one price\
    (e.g., a Framer template, Icon pack, ebook)

    [More about adding products](/add-products)
  </Step>
  <Step title="Provide tax information">
    Variable products allow customers to choose options like size or color on one page, with unique prices, SKUs, and stock levels for each combination (e.g., a shirt). Variable products improve user experience, while simple products are easier to set up.
  </Step>
  <Step title="Setup your payouts">
    Variable products allow customers to choose options like size or color on one page, with unique prices, SKUs, and stock levels for each combination (e.g., a shirt). Variable products improve user experience, while simple products are easier to set up.
  </Step>
  <Step title="Submit for review">
    Variable products allow customers to choose options like size or color on one page, with unique prices, SKUs, and stock levels for each combination (e.g., a shirt). Variable products improve user experience, while simple products are easier to set up.
  </Step>
</Steps>

<Note>
  Review process takes approximately 8 hours, after which you'll receive an email with your store approval or if we need any information from you.
</Note>
